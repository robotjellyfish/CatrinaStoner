## Purpose
This repository preserves my montagalogue and contamimagination in the bedrock of world wide web as a digitally fossilized archive of my mind. It is a persistent, evolving ecosystem of my photography, artwork, published research, my blog, schizophrenic brain vomit and existential poetry.

## Technical Architecture
* **Platform:** GitHub Pages
* **Framework:** Static HTML/CSS/JS (custom implementation)
* **Design Philosophy:** Contamimagination. An iterative collage of visual and textual data displayed in neon gothic horror noir.
* **Maintenance:** Manual indexing, periodic structural shifts, and ongoing content accretion.
